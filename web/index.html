Welcome to Steganography 101

Your goal is to find 3 different secret phrases/"flags", each of which are hidden within images or other files.

Once you find all 3 flags, concatenate them in order and wrap them in CTF_SDaT{} to get the answer

For example, if the three flags were:

  1. flag1
  2. Fl4g2
  3. flag3

then the correct final flag would be CTF_SDaT{flag1Fl4g2flag3}

NOTE: each page (not include this homepage) has exactly one flag. Not every file/image you find will contain a secret message or flag, so poke around until you find it

Select "Flag 1", "Flag 2", or "Flag 3" from the navbar above to get started.
